# Codex Operating Policy

Codex is the local builder/operator.

Rules:

- official ChatGPT auth first
- API key disabled by default
- no browser/session-token hacks
- no hidden mutations
- no bulk rewrite scripts
- use envctl table flow
- use GitKB for memory
- use meta for repo graph
- use RTK safely
- preserve raw logs
- record manual changes

Codex executes through doctrine, not ad hoc patches.
